AudioSyncer

Multicam-Aufnahmen automatisch synchronisieren und als Premiere Pro Projekt exportieren.

macOS 14+ Swift & SwiftUI Keine Dependencies
↓ Source Code auf GitHub Installationsanleitung

So funktioniert's — in 5 Schritten

Audio laden

Externe Recorder-Datei als Audio Master hinzufügen

Kameras laden

Bis zu 6 Video-Dateien per Drag & Drop

Synchronisieren

Ein Klick — der Rest passiert automatisch

Exportieren

Premiere Pro Projekt (.prproj) erstellen

Schneiden

Multicam-Timeline direkt in Premiere Pro öffnen

Features

🔊

Audio-Waveform-Sync

Kreuzkorrelation im Frequenzbereich mit FFT — dieselbe Methode wie PluralEyes. Hardwarebeschleunigt mit Apple Accelerate.

📁

Drag & Drop

Dateien einfach in die Slots ziehen. Unterstützt MP4, MOV, WAV, MP3, AAC und viele mehr.

📊

Waveform-Vorschau

Visuelle Darstellung der Audio-Wellenformen mit Sync-Offset-Anzeige für jede Kamera.

🎬

Premiere Pro Export

Generiert native .prproj Dateien. Öffnet sich direkt in Premiere Pro mit fertiger Multicam-Timeline.

⚡

Auto-Detection

Erkennt Auflösung und Framerate automatisch aus den Videodateien. Unterstützt 23.976 bis 60 fps.

🎨

Premiere Pro Look

Dunkles Interface angelehnt an Adobe Premiere Pro — fühlt sich an wie zu Hause.

Installation

# Repository klonen
git clone https://github.com/fmatsch/AudioSyncerPremierePro.git
cd AudioSyncerPremierePro/AudioSyncer

# Release-Build erstellen
xcodebuild -scheme AudioSyncer -configuration Release build CONFIGURATION_BUILD_DIR=../build

# App starten
open ../build/AudioSyncer.app

Technologie

SwiftUI Native macOS Interface
AVFoundation Audio/Video-Verarbeitung
Accelerate / vDSP Hardware-FFT für Sync
Zero Dependencies Nur Apple Frameworks
